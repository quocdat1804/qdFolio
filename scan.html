SmartScan Project

SmartScan

About the app

SmartScan is a document and customer profile scanning application designed to replace traditional document scanners at VietinBank's branch offices. Processed documents can be synchronized with the bank's server. The application incorporates multiple security layers such as Smart OTP, FaceID, and TouchID, as well as End-to-End Encryption.

‍Things I learned:

  • Working with FaceID
  • Storing in Keychain
  • OTP and end-to-end encryption
  • Using setEditing on views
  • Use camera to scan document
  • Using UIPasteboard to copy to and from the general pasteboard
  • Custom PDF drawing
  • RESTful API, CryptoSwift - For end-to-end encryption
  • iCloud Keychain to store encrypted user data
  • UserDefaults to store user settings
  • Local Database with Realm and MongoDB
  • Tracking if app is in the background & foreground
