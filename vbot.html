VBot Project

VBot

About the app

VBot is a Call - Center system that unifies the customer care service channels of the store / business (phone, Zalo shop, Facebook page) on a single system.

VBot integrates the operator's mobile number as a hotline to the switchboard to make outgoing calls and receive incoming calls using a mobile number with unlimited number of calls.

‍Things I learned:

  • How to use App Store Connect
  • MessageUI, Contacts, ContactsUI, Social – For connecting Facebook Page and Zalo OA
  • CallUI, CallKit, VOIP, WebRTC & SIP
  • Websocket - For Real-time Data Transmission
  • RESTful API, CryptoSwift - For end-to-end encryption
  • MessageUI, XMPP - For chatting
  • iCloud Keychain to store encrypted user data
  • UserDefaults to store user settings
  • Local Database with Realm and MongoDB
  • Better acquainted with Auto-Layout & Texture
  • Tracking if app is in the background & foreground
  • Some CoreMotion
